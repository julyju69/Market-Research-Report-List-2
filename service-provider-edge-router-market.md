<p><strong>The Service Provider Edge Router Market Analysis by types is segmented into:</strong></p>
<p><ul><li>Subscriber Edge Router</li><li>Label Edge Router</li></ul></p>
<p><p>Subscriber Edge Router, also known as a customer edge router, is a type of service provider edge router that connects the customer's network to the service provider's network. It is responsible for routing traffic between the customer's premises and the service provider network. On the other hand, Label Edge Router (LER) is a type of edge router in a Multiprotocol Label Switching (MPLS) network that performs label imposition and disposition functions. It is responsible for forwarding labeled packets based on the labels assigned by the provider edge router.</p></p>
